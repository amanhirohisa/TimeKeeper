# TimeKeeper

## 概要

学会や卒論・修論発表でのタイムキーパー（発表時間を表示しつつ，所定の時間にベル等を鳴らす）ツールです．

## インストール

Java が動作する環境であれば特別な作業は不要です．  
Jar ファイルをダウンロードするだけでよいですが，
ベル等の音声ファイル（.wav ファイル）は別途用意する必要があります．

## 使い方

TimeKeepr.jar ファイルを置いているフォルダで次のコマンドを実行します：  
```java -jar TimeKeeper.jar [-f wav_file] [-s] time1 [time2 time3 ...]```

オプション
- ```-f wav_file``` : ベル音として使用する音声ファイルを指定します．無指定の場合はビープ音になります．
- ```-s``` : ベルを鳴らすまでの時間指定を**秒**単位とします．デフォルトは**分**単位です．
- ```time1``` : 最初にベルを鳴らすまでの時間です．1 以上の整数で指定します．指定された時間経過後にベルを 1 回鳴らします．
- ```time2``` : 2 回目にベルを鳴らすまでの時間です．指定された時間経過後にベルを 2 回鳴らします．
- ```time3``` : ```time1``` や ```time2``` と同様です．4 つ以上指定することもできます．

### 例１（ベル音として bell.wav を用意し，5 分で 1 回， 7 分で 2 回， 10 分で 3 回鳴らす）
```java -jar TimeKeeper.jar -f bell.wav 5 7 10```

### 例2（ベル音として bell.wav を用意し，30 秒で 1 回， 1 分で 2 回， 2 分で 3 回鳴らす）
```java -jar TimeKeeper.jar -f bell.wav -s 30 60 120```
